# Summary

"A-Mem" is an agentic memory system designed to overcome the limitations of rigid, predefined memory structures in [[LLM Agents]]. By applying the [[Zettelkasten method]], the system enables autonomous memory organization through dynamic indexing and linking. When new information is processed, the system creates structured notes and evolves existing memory networks by identifying semantic connections, allowing the agent to adapt its knowledge base over time without manual schema configuration.

# Main Contribution

- Proposed [[A-MEM]], an agentic memory system that enables autonomous memory management and organization.
- Designed an agentic update mechanism featuring automated link generation and [[memory evolution]].
- Demonstrated superior performance across six foundation models compared to existing state-of-the-art baselines.

# Research Problem

Existing memory systems for [[LLM Agents]] rely on rigid, predefined storage structures and fixed workflows, which hinder their ability to adapt to diverse, open-ended tasks. These systems often struggle to organize knowledge dynamically as an agent gains new experiences, leading to poor generalization in long-term interactions. This paper addresses the gap in creating a flexible, universal memory architecture that supports autonomous knowledge evolution rather than static retrieval.

# Method

The [[A-MEM]] architecture is inspired by the [[Zettelkasten method]], focusing on atomic note-taking and flexible linking. The process begins with **Note Construction**, where interactions are converted into structured notes containing timestamps, content, contextual descriptions, keywords, tags, and [[embedding vectors]].

The system then performs **Link Generation** and **Memory Evolution**. Link generation uses semantic similarity to connect new notes to relevant historical memories, forming interconnected "boxes" of knowledge. Memory evolution allows the system to update the contextual representations of existing memories as new information is integrated, enabling the knowledge network to refine its understanding continuously. Unlike [[Retrieval-Augmented Generation]] (RAG) systems that maintain static knowledge bases, A-MEM exhibits agency at the storage and evolution level.
