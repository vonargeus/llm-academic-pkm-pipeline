# Summary

This paper introduces a novel [[semantic loss function]] designed to integrate [[symbolic knowledge]] and [[logical constraints]] directly into [[deep learning]] models. The core idea is to bridge the gap between continuous [[neural output vectors]] and discrete logical statements by deriving a differentiable loss that quantifies how well a network's output satisfies given constraints. The authors demonstrate that this loss function significantly improves performance in [[semi-supervised multi-class classification]] and enhances the ability of [[neural networks]] to predict complex [[structured objects]] like rankings and paths.

# Research Problem

This paper addresses the challenge of effectively integrating [[symbolic knowledge]] and [[logical constraints]] into [[deep learning]] models to improve performance, especially for [[structured output prediction]] and [[semi-supervised classification]]. While [[representation learning]] has seen widespread success, many [[AI]] tasks still benefit from or require classical [[symbolic reasoning]]. Existing [[neuro-symbolic AI]] approaches often either simulate symbolic reasoning in an end-to-end network or embed symbolic knowledge in vector spaces, frequently losing the precise logical meaning of the knowledge due to the need for differentiable models. This paper aims to fill this gap by providing a method for differentiable yet sound logical reasoning that preserves semantic meaning.

# Method

The core of the proposed approach is the [[Semantic Loss Function]], Ls(α, p), which quantifies how close a [[neural network]]'s output vector `p` is to satisfying a given [[propositional logic]] sentence `α`. Unlike prior [[neuro-symbolic AI]] methods that might sacrifice logical precision for differentiability, this work derives the [[semantic loss function]] from first principles, ensuring it precisely captures the meaning of the constraint independent of its syntax.

Formally, the [[semantic loss function]] is defined as proportional to the negative logarithm of the probability of generating a state that satisfies the constraint, when sampling values according to the network's output probabilities `p`. This can be intuitively understood as the "surprise" of obtaining an assignment that satisfies the constraint. The derivation is grounded in a set of intuitive axioms, including [[Monotonicity]] (stricter constraints lead to higher or equal loss) and [[Identity]] (zero loss when the network's output perfectly matches a deterministic state represented by the constraint). A key consequence of these axioms is that the [[semantic loss function]] is semantically equivalent, meaning logically equivalent sentences incur identical loss, reinforcing its independence from syntax.

The [[semantic loss function]] is integrated into standard [[deep learning]] architectures by adding it to the overall loss function during training. For [[semi-supervised learning]], it allows the network to derive a learning signal from vast amounts of unlabeled data by enforcing basic constraints like [[one-hot encoding]] (exactly-one constraint) on outputs, thereby improving confidence in classifications. For [[structured prediction]] tasks, it helps the network learn both the structure of the output space and the classification function within that space by minimizing the loss for complex [[logical constraints]] representing structures like [[total ordering]] or [[valid simple path constraint]].

# Dataset / Experimental Setup

The paper evaluates the [[semantic loss function]] across two main types of tasks: [[semi-supervised multi-class classification]] and [[structured prediction]].

For [[semi-supervised classification]], experiments were conducted on widely used image datasets:

-   [[FASHION]]: A dataset of Zalando's article images, similar in size to [[MNIST]].

For [[structured prediction]], the experiments focused on tasks with highly structured outputs:
-   [[Preference learning]]: Involving [[total ordering]] constraints, where the network must predict a ranking of items, effectively encoding a [[permutation matrix]].
-   [[Path prediction]]: Involving [[valid simple path constraint]] in a graph, where the network must predict a valid simple path between a source and destination.
