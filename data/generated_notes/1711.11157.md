# Summary

This paper introduces a novel, differentiable [[semantic loss]] function that bridges deep neural networks and symbolic reasoning by evaluating how well a network's output satisfies a given set of [[logical constraints]]. Derived from first principles and propositional logic axioms, the loss function measures the "surprise" of obtaining a valid state when sampling from the network's output distribution. The authors demonstrate that integrating this loss function allows neural networks to leverage unlabeled data in [[semi-supervised learning]] via [[one-hot encoding]] constraints, and significantly improves performance on structured prediction tasks such as [[path prediction]] and [[preference learning]].

# Main Contribution

- Derivation of a differentiable [[semantic loss]] function from first principles to bridge neural outputs and [[logical constraints]].
- Application of [[semantic loss]] to [[semi-supervised learning|semi-supervised classification]], achieving near-state-of-the-art results on standard image datasets.
- Enhancement of neural networks' ability to predict complex structured objects like rankings and paths by enforcing logical consistency during training.

# Research Problem

The paper addresses the challenge of integrating symbolic [[logical constraints]] into deep learning models. While deep neural networks excel at perception and representation learning, they struggle to guarantee that their outputs satisfy hard logical rules (e.g., ensuring a multi-class output is strictly one-hot, or that a predicted path in a graph is valid). Traditional neuro-symbolic methods often relax these constraints or map them to vector embeddings, which loses their precise logical meaning. The gap addressed is the lack of a mathematically sound, differentiable loss function that preserves the exact semantics of [[propositional logic]] without destroying the gradient-based optimization of neural networks.

# Method

The authors propose [[semantic loss]], a differentiable loss function $L_s(\alpha, p)$ that evaluates the discrepancy between a neural network's output probability vector $p$ and a [[propositional logic]] formula $\alpha$. The loss is defined as the negative logarithm of the probability of generating a state that satisfies the constraint $\alpha$, assuming the output variables are sampled independently according to $p$:

$$L_s(\alpha, p) \propto -\log \sum_{x \models \alpha} \prod_{i: x \models X_i} p_i \prod_{i: x \models \neg X_i} (1 - p_i)$$

This formulation is derived axiomatically from principles such as Monotonicity (stricter constraints yield higher or equal loss) and Semantic Equivalence (logically equivalent formulas yield identical loss). 

Unlike prior work that relies on syntactic relaxations, [[semantic loss]] is syntax-independent. In [[semi-supervised learning]], the method applies this loss to the "exactly-one" constraint on unlabeled data, forcing the network to make highly confident, single-class predictions. For structured prediction, the loss is applied to complex combinatorial constraints, such as ensuring the output represents a valid simple path in a grid graph or a total ordering (permutation matrix) in [[preference learning]].

# Dataset / Experimental Setup

The method is evaluated across two primary experimental setups:

1. **[[semi-supervised learning|Semi-Supervised Classification]]**: Evaluated on the [[MNIST]], [[FASHION]] (Fashion-MNIST), and [[CIFAR-10]] datasets. The models are trained on a small subset of labeled data combined with a larger set of unlabeled data, where the unlabeled data is regularized using the [[semantic loss]] of the [[one-hot encoding]] constraint.
2. **Structured Prediction**:
   - **[[preference learning|Preference Learning]]**: Learning a total ordering over $n$ items, where the output must satisfy a permutation constraint.
   - **[[path prediction]]**: Predicting a valid simple path from a source to a destination in a grid graph.

# Evaluation Metrics

- **[[accuracy]]**: Used to measure classification performance on the test sets of [[MNIST]], [[FASHION]], and [[CIFAR-10]], as well as the exact match accuracy for structured outputs (paths and rankings).

# Key Results

- **[[semi-supervised learning|Semi-Supervised Classification]]**: Integrating [[semantic loss]] with the "exactly-one" constraint on unlabeled data yields competitive, near-state-of-the-art [[accuracy]] on [[MNIST]], [[FASHION]], and [[CIFAR-10]] compared to purely supervised and other semi-supervised baselines.
- **Structured Prediction**: The network's ability to predict valid structured objects (such as simple paths and total rankings) is significantly enhanced, ensuring the output satisfies the logical constraints far more consistently than standard networks.

# Limitations

A key limitation of [[semantic loss]] is its computational complexity. Computing the loss requires summing over all satisfying states $x \models \alpha$, which is generally NP-hard. While tractable for simple constraints (like one-hot encodings) or compile-time solvable structures (using Sentential Decision Diagrams), scaling this approach to highly complex, dynamic, or very large-scale logical formulas remains a significant computational challenge.
